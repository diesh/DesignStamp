Connect Storage

One-time setup using a free Firebase database so all phones sync live. Takes 2 minutes.

  1. Go to console.firebase.google.com
  2. Create a project (skip analytics)
  3. Click Realtime Database → Create database → choose Start in test mode → Enable
  4. Copy the URL shown — looks like https://your-project-default-rtdb.firebaseio.com
